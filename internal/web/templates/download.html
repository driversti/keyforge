Install KeyForge

Quick Install

Run this command to install keyforge on any supported platform:

curl -sSL {{.ServerURL}}/install.sh | sh

Install + Enroll

Install the binary and enroll this device in one step:

curl -sSL {{.ServerURL}}/install.sh | sh -s -- \ --name "my-device" \ --token YOUR_TOKEN \ --server {{.ServerURL}} \ --accept-ssh
